Game Server Privacy Policy

Last Updated: March 13th, 2025
Our game servers now check IP addresses with IPIntel to detect VPNs and approximate geolocation.

Definitions

Redialing
Automatic reconnection process through the game's launcher to alternative servers
IP Address
Numerical label assigned to your device when connected to a network
UUID (Universal Unique Identifier)
128-bit number used to uniquely identify your game account
HWID (Hardware ID)
Hashed identifier generated from your device's hardware configuration
VPN (Virtual Private Network)
Service that masks your real IP address and location
Server Administrators
Volunteer staff members responsible for maintaining game servers and enforcing rules
Network Operators
Technical personnel maintaining server infrastructure
Server Hosts
Third-party companies providing physical server hardware and network connectivity

Data We Collect

  • IP address and approximate geographic location
  • Hardware ID (HWID)
  • Account UUID and username
  • All in-game actions and communications
  • Explicitly entered user preferences
  • Discord user ID and roles (if you choose to link your Discord account)

Data Access

Internal Teams

Server Administrators

    • Connection histories (IP/HWID/UUID)
    • In-game chat and game logs
    • Moderation action records

Network Operators

    • All Server Administrator access plus:
    • Raw database records
    • Unfiltered connection metadata
    • Server diagnostic streams
    • Persistent storage archives

External Services

Server Hosts

  • Can view: Network traffic patterns, hardware utilization
  • Cannot view: Decrypted game data, personal identifiers
  • Bound by data processing agreements (DPAs)

Security Services (IPIntel)

  • Receives: New connection IP addresses
  • Retention: Immediate deletion after VPN check

Connection Management

VPN Detection

  • New connections are checked via IPIntel service
  • VPN connections are blocked by default
  • Exemptions available through forum request

Redialing Process

  • Automatic server reconnection will:
    • Only connect you to servers within our official network
    • Maintain existing HWID/UUID identifiers
    • Preserve connection history records
  • No data is shared with third-party servers during redialing
  • Does not collect additional personal data

Location Tracking

  • Country data derived from IP addresses
  • Used for:
    • Server allocation analytics
    • Connection quality monitoring
    • Regional rule enforcement

Game Server Privacy Policy

Last Updated: December 25th, 2025
Added Data Retention section, expanded Lawful Basis for Processing, clarified limitations on data erasure requests, and updated Data Transfers and Security Incidents sections for GDPR compliance.

Definitions

Redialing
Automatic reconnection process through the game's launcher to alternative servers
IP Address
Numerical label assigned to your device when connected to a network
UUID (Universal Unique Identifier)
128-bit number used to uniquely identify your game account
HWID (Hardware ID)
Hashed identifier generated from your device's hardware configuration
VPN (Virtual Private Network)
Service that masks your real IP address and location
Server Administrators
Volunteer staff members responsible for maintaining game servers and enforcing rules
Network Operators
Technical personnel maintaining server infrastructure
Server Hosts
Third-party companies providing physical server hardware and network connectivity

Data We Collect

  • IP address and approximate geographic location
  • Hardware ID (HWID)
  • Account UUID and username
  • All in-game actions and communications
  • Explicitly entered user preferences
  • Discord user ID and roles (if you choose to link your Discord account)

Data Access

Internal Teams

Server Administrators

    • Connection histories (IP/HWID/UUID)
    • In-game chat and game logs
    • Moderation action records

Network Operators

    • All Server Administrator access plus:
    • Raw database records
    • Unfiltered connection metadata
    • Server diagnostic streams
    • Persistent storage archives

External Services

Server Hosts

  • Can view: Network traffic patterns, hardware utilization
  • Cannot view: Decrypted game data, personal identifiers
  • Bound by data processing agreements (DPAs)

Security Services (IPIntel)

  • Receives: New connection IP addresses
  • Retention: Immediate deletion after VPN check

Connection Management

VPN Detection

  • New connections are checked via IPIntel service
  • VPN connections are blocked by default
  • Exemptions available through forum request

Redialing Process

  • Automatic server reconnection will:
    • Only connect you to servers within our official network
    • Maintain existing HWID/UUID identifiers
    • Preserve connection history records
  • No data is shared with third-party servers during redialing
  • Does not collect additional personal data

Location Tracking

  • Country data derived from IP addresses
  • Used for:
    • Server allocation analytics
    • Connection quality monitoring
    • Regional rule enforcement

GDPR Specific Info

Your GDPR Rights

EU residents have the right to:

  • Request access to personal data
  • Rectify inaccurate information
  • Erase data under certain conditions
  • Restrict processing of your data
  • Data portability
  • Object to processing

Submit requests to: administrator@goobstation.com

Limitations on Data Erasure

You may request deletion of your personal data. However, under GDPR Article 17, we may refuse erasure requests where data is required for:

  • Moderation and rule enforcement
  • Security and abuse prevention
  • Ban enforcement, including active bans and permanent bans
  • Maintaining moderation history that remains necessary for enforcement or contextual review

Character data and other non-essential gameplay data may be deleted by you at any time, as this data is not required for moderation or enforcement purposes.

Lawful Basis for Processing

Under GDPR, we process data based on:

  • Contractual Necessity: Account data required to provide services
  • Legitimate Interests: Security monitoring, fraud prevention, moderation, rule enforcement, maintaining moderation history, detecting repeat or escalating violations, preventing ban evasion, abuse investigation, and handling appeals. Retention of historical moderation records is necessary to ensure fair, consistent, and effective enforcement across our community.
  • Consent: Optional features like Discord integration

Data Transfers

Data may be transferred outside the EU to:

  • Our hosting providers in the United States
  • Third-party services like Discord and IPIntel

We ensure transfers comply with GDPR through:

  • EU Standard Contractual Clauses (SCCs)
  • Appropriate data processing agreements

Data Retention

We retain different types of data for different purposes:

  • Gameplay and chat logs: Retained for up to 30 days to support moderation review and incident investigation. Logs are automatically deleted after this period unless they are part of an active moderation case.
  • Moderation records: Records such as bans, warnings, and administrative notes are retained as long as they are necessary for moderation, rule enforcement, abuse prevention, and maintaining the integrity of the service. This includes historical records of previous bans and warnings, which help moderators understand patterns of behavior and ensure consistent enforcement.
  • Permanent bans: Treated as ongoing enforcement actions and retained for as long as necessary for moderation and enforcement purposes.
  • Hardware identifiers (HWIDs) and connection identifiers (such as IP addresses): Retained where necessary to enforce bans, prevent ban evasion, and protect the integrity of the service. These identifiers are used solely for enforcement and security purposes, and are not used for marketing, profiling, or advertising.

We do not retain data longer than necessary for its stated purpose.

Security Incidents

We will notify affected EU users within 72 hours of becoming aware of a data breach that risks their rights and freedoms. Affected users will be notified directly where feasible. Public announcement channels, such as our Discord server, may be used as a supplementary notification method.

Contact us at administrator@goobstation.com for inquiries.
